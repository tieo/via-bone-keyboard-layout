# Keymaps

VIA keymap JSONs, one per board. File naming: `<vendor>-<model>-<variant>.json`,
e.g. `keychron-q6-pro-iso.json`.

Each file is a VIA "Save Layout" export. To use:

1. Set OS keyboard layout to plain German (QWERTZ).
2. In VIA / your configurator: **File → Import Layout** → pick the JSON.

To contribute a new board: see [`../docs/physical-layout.md`](../docs/physical-layout.md).
